# Ordinary Girl Meghan / Magical Girl Ava

A story-heavy RPG/Adventure game built in Unreal Engine.

---

## About

*Ordinary Girl Meghan / Magical Girl Ava* (MGA) is an adult action-adventure RPG set approximately 300 years in the future on the colony planet Terridyn. The game follows Meghan, a young woman trying to leave her past behind — and the world that refuses to let her.

**Platforms:** Windows (primary), Linux (secondary), Mac (aspirational)
